# Reel Stock Management System - User Guide

## Software Overview

### English
The Reel Stock Management system is designed for packaging plants to manage paper reel inventory efficiently. It provides comprehensive tracking of reel receipts, consumption, returns, and stock levels. Key features include real-time dashboards, automated label printing, detailed history tracking, and multi-user support with role-based access.

## Getting Started
1. Open browser and navigate to the application URL.
2. Login with provided credentials.
3. Access dashboard for overview.

## Managing Reel Receipts
- Add new reels via "Add Receipt" form.
- Auto-generated reel numbers.
- Print standardized labels.

![Add Receipt Form](screenshots/add_receipt.png)

## Viewing History
- Click reel number to open history in new tab.
- Shows consumption details, GSM, bursting strength.

![History Tab](screenshots/history_tab.png)

## Issuing & Returning Reels
- Issue reels to departments (default "Corrugation Plant").
- Record returns with condition and remaining weight.

## Reports & Analytics
- Stock reports with filters.
- Export to Excel.
- Consumption analytics.

## Troubleshooting
- Enable pop-ups for history.
- Clear browser cache if issues persist.
- Contact IT for server errors.
