Reel Stock Management System

Complete User Tutorial

Software Overview

English Version

The Reel Stock Management System is a comprehensive inventory management solution designed specifically for packaging plants to efficiently track and manage paper reel stock. This system provides real-time monitoring of reel receipts, consumption tracking, returns management, and detailed reporting capabilities. Key features include an intuitive dashboard, automated label printing, complete historical tracking, and multi-user support with role-based access control.

Getting Started - Login

Login Page Instructions

Step 1: Open your web browser and navigate to the application URL (e.g., http://localhost/reelstock)

Step 2: You will see the login page with the following fields:

  • Email Address: Enter your registered email
  • Password: Enter your password

Step 3: Click the Login button

Step 4: Upon successful login, you will be redirected to the main dashboard

Screenshot: Login page with email and password fields

Navigation Menu Overview

After logging in, you will see the main navigation menu at the top of the page:

  • Dashboard - Main overview page with statistics and charts
  • Reel Receipt - Add new paper reels to inventory
  • Reel Issue - Issue reels to departments
  • Reel Stock Report - View current stock and generate reports
  • Paper Quality - Manage paper quality types
  • Suppliers - Manage supplier information
  • Logout - Sign out of the system

Dashboard Page

Main Dashboard Overview

The Dashboard is your main hub for monitoring reel stock status and key metrics.

Dashboard Components:

Summary Cards

  • Total Reels: Shows total number of active reels in stock
  • Total Weight: Displays combined weight of all reels
  • Low Stock Alerts: Number of reels below minimum threshold
  • Recently Added: Latest reel receipts

Stock Chart

  • Visual representation of stock levels by paper quality
  • Color-coded bars for easy identification
  • Interactive tooltips showing exact quantities

Recent Activity

  • List of recent reel receipts and issues
  • Shows date, reel number, and action type
  • Click any item to view details

Low Stock Warnings

  • Red-highlighted reels needing immediate attention
  • Shows current balance and recommended reorder quantity

How to Use Dashboard:

  • Refresh Data: Click the refresh button to update all metrics
  • Filter by Date: Use date selector to view specific time periods
  • Export Data: Click export button to download dashboard data as Excel
  • View Details: Click on any reel number or activity item to see full information

Reel Receipt Page

Adding New Reels to Inventory

The Reel Receipt page allows you to add new paper reels to your inventory system.

Page Layout:

  1. Add Receipt Form (Left Panel)
  2. Existing Receipts List (Right Panel)
  3. Print Labels Button (Top Right)

Form Fields Explained:

Field Name Description Required
Receiving Date Date when reel was received Yes
Paper Quality Type/quality of paper Yes
Supplier Company that supplied the reel Yes
Reel Size Dimensions of the reel Yes
Net Weight Actual weight of the reel Yes
GSM Grams per square meter Yes
Bursting Strength Paper strength measurement Yes
Rate per KG Cost per kilogram Yes
QC Status Quality control status Yes
Remarks Additional notes No
Screenshot: Add Receipt form with all fields filled

Step-by-Step Process:

Step 1: Click on the Reel Receipt menu item

Step 2: Fill in all required fields in the form:

  • Select Receiving Date from calendar
  • Choose Paper Quality from dropdown
  • Select Supplier from dropdown
  • Enter Reel Size (e.g., "32" for 32 inches)
  • Enter Net Weight in kilograms
  • Enter GSM value
  • Enter Bursting Strength
  • Enter Rate per KG
  • Select QC Status (Approved/Pending/Rejected)

Step 3: Click Save Receipt button

Step 4: System will:

  • Generate automatic reel number (e.g., "RL2026000043")
  • Save the receipt to database
  • Add the new reel to the list on the right

Step 5: Print labels by clicking Print Labels

Managing Existing Receipts:

  • View History: Click any reel number to open detailed history in new tab
  • Edit Receipt: Click the edit icon next to any receipt
  • Delete Receipt: Click the delete icon (with confirmation)
  • Search: Use the search box to filter reels by number (without "RL" prefix)

Label Printing:

  1. Select reels by checking the boxes next to reel numbers
  2. Click Print Labels
  3. Print dialog will open with formatted labels
  4. Labels include: company name, reel number, quality, size, weight, GSM, bursting strength
  5. Print on standard label paper (95mm × 96mm)
Screenshot: Print labels dialog with selected reels

Reel Issue Page

Issuing and Returning Reels

The Reel Issue page manages the distribution of reels to different departments and tracks returns.

Page Sections:

  1. Issue Reel Form - For issuing reels to departments
  2. Return Reel Form - For recording returned reels
  3. Issue History - List of all issued reels

Issue Reel Form Fields:

Field Name Description Default Value
Reel Number Auto-suggest reel from available stock Search and select
Issue Date Date of issue Today's date
Quantity Issued Weight to issue in KG Enter manually
Issue To Department receiving the reel "Corrugation Plant"
Remarks Additional notes Optional
Screenshot: Issue Reel form with default "Corrugation Plant" value

How to Issue a Reel:

Step 1: Click in the Reel Number field

Step 2: Type reel number (without "RL") or select from dropdown

Step 3: Verify the reel details that appear

Step 4: Enter Quantity Issued in kilograms

Step 5: Change Issue To if needed (defaults to "Corrugation Plant")

Step 6: Add Remarks if necessary

Step 7: Click Issue Reel

Step 8: System updates stock and adds to issue history

Return Reel Form Fields:

Field Name Description Notes
Reel Number Select previously issued reel Shows issued reels only
Return Date Date of return Today's date
Remaining Weight Weight returned in KG Must be ≤ issued quantity
Return To Location/department Enter manually
Condition Reel condition status Good/Damaged/Partial
Remarks Additional notes Optional
Screenshot: Return Reel form with condition selection

Reel Stock Report Page

Viewing and Analyzing Stock

The Reel Stock Report page provides comprehensive views of current inventory and consumption patterns.

Report Features:

Advanced Filters

  • Filter by Paper Quality
  • Filter by Reel Size
  • Filter by Balance Weight range
  • Date range filters

Stock Table

  • Shows all available reels
  • Columns: Reel No, Quality, Size, Original Weight, Balance Weight, Status
  • Color-coded status indicators

Export Options

  • Export to Excel
  • Export to PDF
  • Print report

Using Filters:

  1. Quality Filter: Select specific paper quality from dropdown
  2. Size Filter: Enter reel size or select from common sizes
  3. Weight Filter: Set minimum and maximum balance weight
  4. Apply Filters: Click Apply Filters button
  5. Reset: Click Reset to clear all filters
Screenshot: Stock report page with filters applied

Viewing Reel History:

  1. Click on any Reel Number in the stock table
  2. A new tab will open showing complete reel history:
    • Reel Information: Basic details, GSM, bursting strength
    • Receipt History: When and how it was received
    • Issue History: All issuances with dates and quantities
    • Return History: All returns with conditions
    • Running Balance: Current balance calculation
Screenshot: Reel history tab showing complete lifecycle

Exporting Reports:

  1. Apply desired filters
  2. Click Export to Excel for spreadsheet format
  3. Click Export to PDF for document format
  4. Click Print for hard copy

Troubleshooting Guide

Common Issues and Solutions

Login Problems:

  • Issue: Cannot login
  • Solution: Check email/password, contact administrator for reset

Page Loading Issues:

  • Issue: Pages load slowly or show errors
  • Solution: Clear browser cache, check internet connection

History Not Opening:

  • Issue: Reel history doesn't open in new tab
  • Solution: Enable pop-ups for this site in browser settings

Print Issues:

  • Issue: Labels don't print correctly
  • Solution: Check printer settings, use correct label size (95mm × 96mm)

Data Not Saving:

  • Issue: Forms don't save data
  • Solution: Check all required fields are filled, ensure internet connection

Export Problems:

  • Issue: Cannot export reports
  • Solution: Check browser download settings, try different format
